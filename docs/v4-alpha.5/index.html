SuikaはオリジナルBootstrapテーマです

SuikaはBootstrapテーマの一つですが、以下の特徴を持っています。

Easy to Start

SuikaはBootstrapを元に製作されているため、非常に高い互換性を持っています。マークアップに関する規約はほとんど変わりません。

Replace Bootstrap

既にBootstrapを使って作成したウェブサイトがある場合は、CSSを置き換えるだけで簡単にSuikaを利用できます。

Open Source

作成に使用したSASSファイルは全て公開されています。変数の定義ファイルを変更することで自分好みの設定に変更することも可能です。

Japanese WebFont

日本語表示の美しさを追求し、fork元の"Honoka"による調整に加え、日本語ウェブフォントとしていろは角クラシックをJIS第2水準漢字までにサブセット化した上で使用しています。

Suikaは日本語を美しく表示します

英数字部分には「M+ p」を、日本語部分には「いろは角クラシック」を使用しています。ウェイト(太さ)は light （ライト） / regular（レギュラー） / bold（ボールド） の3種類のみ使用しています。
